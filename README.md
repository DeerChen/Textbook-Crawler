## Introduction

自用，下载[人民教育出版社](https://jc.pep.com.cn)教材。

支持正版，请勿传播，谢谢。

## Installation

```bash
pip install json5 requests Pillow
```

## Usage

```bash
python main.py 信息技术选修二 1452001132201 136 # 书名 ID 总页数
python main.py # 需自行先在`textbooks.jsonc`中添加目标书籍信息
```
